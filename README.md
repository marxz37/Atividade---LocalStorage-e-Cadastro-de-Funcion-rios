# Cadastro de Funcionários e Login com Perfil de Acesso

Aplicação web desenvolvida para a prática de manipulação do DOM e uso do LocalStorage em JavaScript. O sistema permite o cadastro dinâmico de funcionários, exibição em formato de cards e persistência dos dados no navegador.

O objetivo principal desta etapa foi evoluir o sistema anterior de Cadastro de Funcionários (baseado em localStorage), implementando uma camada de segurança que inclui autenticação de usuários, diferenciação de perfis de acesso (Administrador e Funcionário) e personalização da interface de acordo com o usuário logado.
