# Cadastro de Funcionários

Aplicação web desenvolvida para a prática de manipulação do DOM e uso do LocalStorage em JavaScript. O sistema permite o cadastro dinâmico de funcionários, exibição em formato de cards e persistência dos dados no navegador.

A atividade será a base para evoluções futuras, como criação de uma tela de login ou controle de acesso.
